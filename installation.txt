1.  Install VLC with hardware acceleration: https://www.raspberrypi.org/forums/viewtopic.php?f=66&t=195221

2.  Set static IP: https://www.raspberrypi.org/forums/viewtopic.php?t=191140+

3.  Enable SSH and auto script start on startup
    Enable ssh in desktop -> preferences -> raspberry config -> interfaces -> ssh
    sudo nano /home/pi/.config/lxsession/LXDE-pi/autostart, add before screensaver:
    sudo python3 /home/pi/Scripts/start.py &
4.  Copy over scripts

5.  Install dependencies:
sudo apt-get install python3-pyqt4 libxml2-dev libxslt-dev
sudo pip3 install tornado psutil lxml purl
(for unit tests: pytest)

6.  Make coap client for lighting controll:
sudo apt-get install build-essential autoconf automake libtool
git clone --recursive https://github.com/obgm/libcoap.git
cd libcoap
git checkout dtls
git submodule update --init --recursive
./autogen.sh
./configure --disable-documentation --disable-shared
make
sudo make install

7.  Disable screensaver: sudo apt-get install xscreensaver -> menu preferences -> screensaver -> disable

8.  Enable overclock/sound via hdmi: sudo nano /boot/config.txt
	hdmi_group=1
	hdmi_mode=19
	arm_freq=1350
	core_freq=500
	over_voltage=4
	disable_splash=1
	gpu_mem=320
	force_turbo=1
	boot_delay=1
